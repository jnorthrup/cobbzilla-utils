static <T> List<List<T> > org.cobbzilla.util.collection.ListUtil.permutations ( List< List< T >>  collections)
static

Combines several collections of elements and create permutations of all of them, taking one element from each collection, and keeping the same order in resultant lists as the one in original list of collections.

Input = { {a,b,c} , {1,2,3,4} } Output = { {a,1} , {a,2} , {a,3} , {a,4} , {b,1} , {b,2} , {b,3} , {b,4} , {c,1} , {c,2} , {c,3} , {c,4} }

Parameters
collectionsOriginal list of collections which elements have to be combined.
Returns
Resultant collection of lists with all permutations of original list.
